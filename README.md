# CSC648 Assignment 3 – MDP Modeling and Policy Optimization

## Overview
This project implements a Frozen Lake Markov Decision Process (MDP) and applies key reinforcement learning algorithms:

- Policy Evaluation
- Policy Improvement
- Value Iteration

The environment is a stochastic 4×4 grid with slippery transitions.


## Key Concepts

- Markov Decision Processes (MDPs)
- Value Function \(V(s)\)
- Action-Value Function \(Q(s,a)\)
- Bellman Equations
- Stochastic environments

## Results

- Value function propagation demonstrated over 10 iterations
- Improved policy derived from Q-values
- Optimal policy computed via value iteration
